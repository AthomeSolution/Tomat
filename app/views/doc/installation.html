Requirements

  • Node.js
  • Android SDK with tools, platform-tools and lastest platform installed
  • Apache Ant 1.8 or later
  • Xcode (for iOS only)

Instructions

All commands are to be executed in a shell. On Linux they might need to be prefixed with sudo (e.g. with npm -g). On Windows you can use Console2 if you want a better console.

Ionic & Cordova

In order to build applications you will need to install Apache Cordova.

npm install -g cordova

At least you need the Ionic command line utility. It uses gulp as building helper.

npm install -g gulp ionic

Build your Android application

Now you can create a new Cordova project. For a project called demo with:

ionic start demo

A new directory as been created for your project. For the next steps you have to go into it:

cd demo

The next step is to tell Ionic that it have to handle Android:

ionic platform android

You should see a new directory in platform called android. It contains an Android native version of your application. You can import this project in your preferred IDE for debugging purpose. Keep in mind that this project will be overridden any tiome your build your application. If you will to do it you can use ionic prepare android to refresh the project.

To build and test your project:

ionic build android
ionic emulate android

Buid your iOS application

Ionic manage the iOS emulator with a CLI application called ios-sim. You will need to install it:

npm install -g ios-sim

Short version, same commands as for Android:

ionic platform ios
ionic build ios
ionic emulate ios

Further information

Fast Android emulator

By default Ionic uses the most up-to-date emulator, for now android-19. You can speed it up by using hardware acceleration. Note that it only works for Intel CPU on Windows.
